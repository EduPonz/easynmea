Developer Documentation
=======================

This section contains all the design documents of *GNSS Interface*.
It is meant to gather all technical documentation so that contributors to the project can understand the reasoning
behind the current implementation, as well as document the designs for their contributions to the library.
Please refer to the `Contributing Guidelines <https://github.com/EduPonz/gnss_interface/blob/main/CONTRIBUTING.md>`_ if
you are considering contributing to *GNSS Interface*.
